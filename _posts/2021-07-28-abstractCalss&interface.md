| 추상클래스                          | 인터페이스                                    |
| ----------------------------------- | --------------------------------------------- |
| 클래스                              | 클래스X, 명세 O                               |
| 변수, 상수                          | 상수(static)                                  |
| 구현된 메소드, 추상 메소드          | 추상 메소드, default/static 메소드(java8부터) |
| 단일상속                            | 다중상속                                      |
| 클래스상속("is a")                  | "is a"로 표현 어려운 관계(공통점 O)           |
|                                     |                                               |
| 공통점: 둘다 객체를 생성할 수 없다. |
